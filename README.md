# Lab 3
CMPINF401 - Loops, Methods & String Processing

## Build

### Maven
Project builds are configured via [Maven](https://maven.apache.org).
Eclipse can also be used to run this project.

To build, run 
```
 mvn package
```

## Usage

### String Reverse
To run
```
mvn exec:java@reverse
```

### Password Generator
To run
```
mvn exec:java@password
```
